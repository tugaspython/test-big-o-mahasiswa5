# Tugas : Pencarian Pasangan Angka Target (Two Sum)

## Deskripsi
Diberikan sebuah daftar (list) berisi angka integer dan sebuah angka target. Buatlah sebuah fungsi bernama cari_pasangan(arr, target) yang mengembalikan nilai True jika terdapat dua angka berbeda di dalam list tersebut yang apabila dijumlahkan hasilnya sama dengan angka target. Jika tidak ditemukan, kembalikan False.

Batasan (Constraints):
Panjang list dapat mencapai hingga 50.000 angka.
Fungsi harus berjalan seefisien mungkin. Batas waktu eksekusi maksimal adalah 2 detik.
